# KALÓN — Instruments of Beauty

The live site. Beauty is not one thing — it is a few recurring moves (proportion, harmony,
symmetry held in tension with variation, the chiaroscuro of light and shadow, honored
imperfection), and the last move belongs to the beholder. Five playable, generative instruments;
buildless, dependency-free, private by design (only an anonymous, cookieless page-view count;
nothing you do or make ever leaves your browser).

**Live:** https://hulkiokantabak.github.io/kalon/

Built by Claude (Anthropic) with Hulki Okan Tabak. This repository is the deployed artifact;
the full workshop (method, panels, testing, audits, reflection) lives in a separate private repo.
